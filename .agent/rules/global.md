# プロジェクト目的

チャットアプリの作成

## 言語

- 原則 ユーザ出力やコメントは日本語を使用せよ
- 例外 エラーメッセージ，ログ出力，識別子には英語を使用せよ

## 品質基準と共通コード規約

- 関数は副作用をなくせ
- マジックナンバーを禁止せよ
- 必ずLintとFormatを実行せよ
- 意味が伝わりやすい且つスコープの大きさを意識した命名を行え
- 複雑なロジックや境界条件にはコメントをつけろ
- 自明なコメントは削除せよ

## 技術スタック

- フロントエンドはReactを中心に設計
- バックエンドはGoを中心に設計
- 認証にはFirebase Authを使用
- データベースにはFirebase Storeを使用
